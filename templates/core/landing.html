Frequently Asked Question

The most common questions you can ask about how UI Stack works and what it can do for you.

How do I register on UI Stack?

Currently, UI Stack is still in the development phase. However, you can register filling out the waitlist form and you will receive an email when UI Stack is live.

Is it free to post on UI Stack?

Absolutely! All you have to do is register with your email.

Is it only for UI students?

The focus of UI Stack is to foster great experiences with the students of the University of Ibadan (UI) as such only UI students are allowed to post any content. However, as a non-UI student, you can still view posts on UI Stack.

Can the info in Stack Time Capsule be verified?

UI Stack uses a voting system in which the community decides whether information posted is legit. Also, a post is tagged with verified badge if upvoted by a lot of members in the UI Stack Community

What is a stack?

A stack refers to your niche. What you at good at. What activty makes you feel good. At UI Stack, a student can have as many stacks as they wish.

How can I meet with people of similiar stacks?

By default, UI Stack will show you the posts of UI students with similiar stacks as yours. Also, you can decide to follow anyone and see their posts.
